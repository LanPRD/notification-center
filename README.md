# Notification Center

## Estrutura do Projeto

- `core/` - Classes base e tipos utilitários
- `domain/` - Entities e contratos de repositórios
- `application/` - Casos de uso (regras de aplicação)
- `infra/` - Implementações (database, HTTP)
  - `database/` - Prisma e repositórios
  - `http/` - Controllers, DTOs, decorators
